# Personal Prompt System

A personal repository for organizing prompt engineering assets.

## Structure

- `project-instructions/` — Core project-level instructions.
- `prompt-corpus/` — Collection of notable prompts.
- `skills/` — Reusable prompt-analysis and prompting skills.
- `references/` — Source notes and index files.
- `tests/` — Prompt quality and analysis test notes.
- `archive/` — Archived drafts and deprecated materials.

## Goal

Build and maintain a personal AI prompt system for research, analysis, and iterative improvement.
