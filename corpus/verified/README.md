# Verified Prompt Corpus

This folder stores prompt samples that have passed a stricter source-evidence intake before pattern extraction.

Each sample should include:

- `meta.md`: source, verification strength, evidence, domain, and curation notes
- `content.md`: short compliant excerpts, structure summary, decomposition notes, and reuse boundaries

Rules:

- Prefer official or source-traceable prompts.
- Do not copy long prompt text.
- Keep copyright and licensing uncertainty visible.
- Treat pattern claims as provisional until multiple verified samples support them.
- Do not modify the existing `prompt-corpus/`, `skills/`, or `specs/` tracks from this folder.
