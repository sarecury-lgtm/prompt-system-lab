# Regret Risk Scorecard

Score only non-excluded products.

| Dimension | Score | Evidence |
|---|---:|---|
| Delivery-included unit price | 0-3 |  |
| Source transparency | 0-3 |  |
| Seller reliability | 0-3 |  |
| Review trust | 0-3 |  |
| Product fit | 0-3 |  |
| Meat-specific risk | 0-3 |  |

## Anchors

- 3: strong evidence and low risk
- 2: usable evidence with minor uncertainty
- 1: weak evidence or meaningful concern
- 0: missing evidence, high risk, or exclusion-level issue

## Judgment

- 15-18: buy
- 10-14: hold / compare alternatives
- 0-9: avoid

Use judgment when one dimension has a severe red flag. A low smell/delivery score can override a good price.
